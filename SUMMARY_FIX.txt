===============================================================================
TEST MANUEL (5 MIN)
===============================================================================

1. Lancer l'UI:
   > streamlit run ui/main.py

2. Sélectionner 2 stratégies (ex: bollinger_atr + rsi_reversal)

3. Vérifier:
   ✅ Deux sections distinctes "📋 Stratégie 1/2" apparaissent
   ✅ TOUS les paramètres de chaque stratégie sont affichés
   ✅ Valeurs par défaut visibles dans les sliders

4. Modifier un paramètre de chaque stratégie

5. Recharger (F5) → vérifier que les valeurs sont persistées

6. Lancer un backtest simple → vérifier que les résultats utilisent
   les paramètres modifiés (≠ defaults)
