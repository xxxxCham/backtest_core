## ✅ Fix appliqué

### Modification : Ignorer les hints si type de stratégie détecté

**Fichier** : `agents/strategy_builder.py`

**Code AVANT** :
```python
# Toujours extraire les hints
hinted_symbol, hinted_timeframe = _find_objective_market_hints(...)
```

**Code APRÈS** :
```python
# Ignorer les hints si type de stratégie détecté
if not detected_strategy_type:
    # Pas de type détecté : utiliser les hints pour guider
    hinted_symbol, hinted_timeframe = _find_objective_market_hints(...)
    logger.info("using hints, symbol=%s, timeframe=%s", hinted_symbol, hinted_timeframe)
else:
    # Type détecté : IGNORER les hints hardcodés
    hinted_symbol = None
    hinted_timeframe = None
    logger.info("strategy_type=%s → IGNORING hints (prioritize intelligent ranking)", detected_strategy_type)
```

**Résultat** :
- Objectif = "[Momentum] sur 0GUSDC en 1h..."
- Type détecté = `momentum`
- Hints ignorés → Pas de "préférence 0GUSDC"
- LLM reçoit seulement : "Tokens prioritaires : APT, AVAX, SOL"
- **LLM choisit : APTUSDC 30m** (optimal pour momentum)
